The Booktoria Redesign

The Booktoria Study

Booktoria is a bookstore website offering both new and secondhand books, including a thrift/resale section. While the site had a functional core browsing, filtering, and checkout it lacked key elements that build buyer trust and support discovery.

As a self-directed redesign project, I audited the site, identified critical UX gaps, and redesigned four core screens to improve product trust and discovery

My role

UX/UI Design (research, wireframing, high-fidelity design)

Tools

Figma

Platforms

Web: Desktop + Mobile

The Problem

Booktoria had a functional buying flow, but users had no way to evaluate whether a book was worth buying there were no descriptions or reviews on product pages.

The homepage offered no starting point for discovery, forcing users to search blindly with no bestsellers or featured picks.

These weren't cosmetic issues they sat directly on the path to purchase, affecting whether a visitor converted into a buyer.

Desktop Experience

Optimizing for deep discovery and detailed comparison.

Homepage

Added a Bestsellers/Featured section to give users an immediate starting point for discovery instead of relying on blind search.

Search & Browse

Introduced filtering by author and stock availability, plus rating-based sorting to help users narrow results faster.

Detail Page

Added an expandable description and reviews so users can confidently evaluate books before purchasing.

Cart

Added a clear price breakdown to improve pricing transparency and reduce cart abandonment.

Mobile Experience

Designed a mobile version so the same trust-building improvements carry across devices.

✓ Key improvement

Booktoria lacked the information users needed to make confident purchasing decisions. My redesign addressed these gaps by adding reviews, book descriptions, bestseller recommendations, and transparent pricing to improve trust, discovery, and the overall user experience.
